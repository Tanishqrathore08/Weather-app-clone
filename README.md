Premium Weather App
🚀 Overview

This is a modern Weather Web Application that provides real-time weather data and a 7-day forecast for any city.

It features a clean, premium UI with smooth animations and glassmorphism design, along with accurate weather data powered by the Open-Meteo API.

💡 Why I Built This

Most weather apps are either too basic or visually outdated.

I built this project to:

Practice working with real-world APIs
Improve my UI/UX design skills
Create a premium-looking web application
Understand how frontend apps handle dynamic data
⚙️ How It Works
🔍 Search System
User enters a city name
The app fetches location + weather data from the API
🌡️ Current Weather
Displays:
City name
Temperature
Weather condition
Min/Max range

📅 7-Day Forecast
Shows daily weather data including:
Temperature (high/low)
Weather icons
Precipitation
📡 API Integration
Uses Open-Meteo API for:
Weather data

Forecast data
